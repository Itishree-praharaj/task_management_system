# task_management_system
A task management app built using the MERN stack (with **MySQL** as the database). It includes task creation, editing, drag-and-drop task movement, and user authentication.

- Flowbite-React
- React Toastify

### 🔹 Backend
- Node.js
- Express.js
- MySQL

- JWT Authentication
- Bcrypt.js
- Cookie-Parser
